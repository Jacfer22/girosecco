## Genera tour (JPG + MP4)

```bash
npm run social:tour
```

Output: `social/tour/out/instagram/` (gitignored, privato)

## Cosa contiene

- **9 JPG** — tour step-by-step con didascalie in foto
- **1 MP4** — tour completo ~36s
- **3 MP4** — reel per mascotte (~12s)
- **DIDASCALIE.md** — testo da incollare sotto ogni post
